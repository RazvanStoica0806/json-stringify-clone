# JSON.stringify Clone  
Re-implementation of `JSON.stringify` for learning and interview practice.  

## Features  
- Supports primitives, arrays, objects  
- Handles undefined / functions correctly  
- Detects circular references  
- Written in TypeScript  
- Fully unit tested  

## Run Tests  
```bash  
npm install  
npm test  
```  

## Notes  
This implementation does not support replacer, spacing, or BigInt.
